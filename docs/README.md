# Documentation

This folder contains all documentation for the Engine project.

## Structure

- `architecture/` - System architecture and design documents
  - `architecture.md` - High-level architecture overview
  - `system_design.md` - Detailed system design specifications
  - `diagrams/` - Visual diagrams and flowcharts (add .png, .svg, or .mermaid files here)

## Updating Documentation

When updating the architecture or system design:

1. Update the relevant `.md` files with new information
2. Add diagrams to the `diagrams/` folder
3. Update the "Last updated" timestamps
4. Ensure cross-references remain accurate

## Contributing

- Use clear, concise language
- Include code examples where helpful
- Keep diagrams simple and readable
- Update this README when adding new documentation sections

## Tools

- Diagrams can be created with:
  - Draw.io (export as SVG)
  - Mermaid (for text-based diagrams)
  - Any vector graphics editor
